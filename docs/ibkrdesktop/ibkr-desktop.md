# IBKR Desktop

## 概述

**IBKR Desktop** 是盈透证券（Interactive Brokers）推出的新一代桌面交易平台。它从零开始重新设计了整个 UI 框架，把 TWS 中经过实战检验的常用工具，与一系列原创新功能（**期权价格网格 Option Lattice**、**多列排序筛选器 Screeners with MultiSort** 等）融合在一起。熟悉与新鲜的结合，加上盈透一贯卓越的订单执行能力，让 IBKR Desktop 既**界面清爽、响应迅速、易于上手**，又**功能强劲不妥协**。

IBKR Desktop 的导航逻辑经过重新打磨，学习曲线非常平缓——**买一只单股和搭一个复杂的多腿期权策略，操作一样简单**。

!!! note "覆盖范围"
    通过 IBKR Desktop，你可以在全球 160 多个市场上交易**股票、期权、期货、外汇**等品种。

!!! note "对照 TWS"
    IBKR Desktop 是盈透的**新一代桌面平台**，与老牌的 [Trader Workstation（TWS）](../tws-manual/README.md) **长期共存**。TWS 拥有 500+ 章节覆盖深度专业功能，IBKR Desktop 则聚焦核心功能、追求简洁界面——本模块的~160 章都是 Desktop 上的内容。

## 下载软件

按以下步骤获取 IBKR Desktop 桌面客户端：

1. 打开 IBKR Desktop [下载页](https://www.interactivebrokers.com/en/trading/ibkr-desktop-download.php)。
2. 点击 **Download for Windows** 按钮，下载适用于 Windows 系统的安装包。
3. 浏览器开始下载安装程序。下载完成后，**双击**安装文件并按屏幕上的提示完成安装。

!!! note "界面位置"
    **Download for Windows** 按钮位于 IBKR Desktop 下载页的中上部，旁边会标注当前的 Latest 稳定版版本号。

!!! tip "Mac / Linux 用户"
    截至本文档最后更新，IBKR Desktop 官方仅提供 **Windows** 版本。如果你使用 macOS 或 Linux，可以考虑以下替代：
    - **macOS**：运行 Windows 虚拟机（Parallels / VMware Fusion）安装 Windows 版 IBKR Desktop，或使用网页版 [Client Portal](../client-portal-platform.md)
    - **Linux**：通过 Wine / 虚拟机运行 Windows 版，或使用网页版 Client Portal
    - **始终可用**：网页版 [Client Portal](https://ibkrguides.com/clientportal/home.htm) 不依赖操作系统

## 关键要点

- **新一代桌面平台**：从零设计的 UI，融合 TWS 的成熟工具与 Desktop 的原创新功能（Option Lattice / MultiSort Screeners）。
- **覆盖 160+ 全球市场**：股票、期权、期货、外汇等。
- **学习曲线平缓**：单股买入与多腿期权策略操作路径一致。
- **不是 TWS 的替代品**：与 TWS 长期共存；TWS 用户无需迁移，新用户可直接从 Desktop 起步。
- **目前仅 Windows**：官方客户端只支持 Windows 系统，其它系统可考虑 Client Portal 网页版。
