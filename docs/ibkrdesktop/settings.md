## 概述

**设置（Settings）** 是 IBKR Desktop 内置的全局配置入口，包含**所有可个性化配置项**的统一管理面板——快捷键、通知、显示偏好、布局设置等都从这里进入。

设置面板的入口在 IBKR Desktop 主界面右上角的**齿轮图标**。

## 打开设置面板

1. 点击 IBKR Desktop 主界面右上角的 **Settings（设置）** 图标（齿轮形状）
2. 在弹出的设置窗口中切换各分类标签进行配置
