## 概述

**IBKR Desktop** 是盈透证券（Interactive Brokers）推出的**新一代桌面交易平台**——它从零重新设计了 UI 框架，把 TWS 中经过实战检验的常用工具，与一系列原创新功能（如期权价格网格 Option Lattice、多列排序筛选器 Screeners with MultiSort）融合在一起，目标是为追求**简洁界面**的现代交易者提供一个**清爽、响应迅速、易于上手**的平台。

如果你以前用过 TWS，第一次打开 IBKR Desktop 会明显感觉：**界面更聚焦、面板更少、默认布局就能上手**，但下单与组合管理核心功能并没有妥协。

!!! note "覆盖范围"
    通过 IBKR Desktop，你可以在全球 160 多个市场上交易**股票、期权、期货、外汇**等品种。

!!! note "对照 TWS"
    IBKR Desktop 是盈透的**新一代桌面平台**，与老牌的 [Trader Workstation（TWS）](../tws-manual/README.md) **长期共存**。
    - **TWS**：专业级全功能平台，500+ 章节覆盖深度功能
    - **IBKR Desktop**：精简版新一代平台，~160 章覆盖核心功能（本模块）
    TWS 用户无需迁移，新用户可直接从 Desktop 起步——两者共用同一盈透账户、同一合约数据库、同一成交体系。

## 安装后第一步

下载并安装完 IBKR Desktop 后，按以下顺序快速进入状态：

1. **登录账户**：使用你的盈透账户（**IBKR Account**）用户名 / 密码登录，首次登录会引导你完成一次性设置（如交易密码）。
2. **熟悉默认布局（Layout）**：登录后会进入一个简洁的**单一工作区**。IBKR Desktop 的核心理念是"一个 Layout = 一个工作场景"，所有面板（Panes）都在这个 Layout 内自由组合。
3. **找一个产品**：使用主界面**左上角**的搜索框（合约搜索 Contract Search）输入关键字，例如 "AAPL" 或 "Apple"，打开产品的报价窗口（Quote）。
4. **下第一笔测试单**：在报价窗口中点击 **Buy** 或 **Sell** 按钮，弹出下单面板——先在模拟账户（Paper Trading Account）上熟悉流程。
5. **管理账户设置**：点击**右上角账户头像（Account Avatar）**，进入账户管理菜单（充值、行情订阅、报表、退出等）。

!!! note "界面位置"
    默认 Layout 通常包含：**顶部搜索框**（左上角）+ **Layout 切换器**（顶部）+ **账户头像**（右上角）+ **主工作区**（中央，可放置投资组合 / 自选列表 / 图表等面板）。

## 关键要点

- **新一代 UI**：从零设计，比 TWS 更简洁、更聚焦。
- **核心场景**：单一 Layout 内自由组合面板（Portfolio / Watchlist / Chart / Quote 等）。
- **入门三件套**：合约搜索 → 报价窗口 → 下单面板——三步完成"找到一个产品并下单"。
- **与 TWS 并存**：同一账户、同一数据源，TWS 用户不需要迁移。
- **目前仅 Windows**：官方客户端只支持 Windows 系统；Mac / Linux 可考虑 Client Portal 网页版。
