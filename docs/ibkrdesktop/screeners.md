## 概述

**筛选器（Screeners）** 是 IBKR Desktop 内置的"按条件搜索标的"工具集——按财务指标、技术指标、价格区间、波动率等条件批量筛选股票、ETF、期权或债券。它对应 IBKR Desktop 左侧导航的 **Screeners** 入口。

## 操作入口

筛选器从 IBKR Desktop 左侧导航面板的 **Screeners** 图标进入。打开后顶部会显示 5 个产品分类标签页：

- **US Stocks**（美股）
- **Asia Stocks**（亚太股票）
- **EUR Stocks**（欧洲股票）
- **ETFs**（交易所交易基金）
- **Options**（期权）

每个标签页内可选择对应的交易所（exchange），并选用三种筛选器类型之一（详见 [筛选器概览](screeners-overview.md)）。
