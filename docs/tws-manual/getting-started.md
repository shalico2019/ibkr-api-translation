# 入门概览

## 概述

[Trader Workstation（TWS）](https://www.interactivebrokers.com/en/trading/tws.php) 是盈透证券（Interactive Brokers）的旗舰桌面交易平台，专为需要交易多种产品、对功能与灵活性有较高要求的有经验活跃交易者设计。通过 TWS，你可以在一个统一平台上交易**股票、期权、期货、外汇、债券和基金**，覆盖全球 160 多个市场。

TWS 同时提供两种界面风格：

- **Mosaic 界面**：默认现代界面，所有交易工具集中在一个窗口内，便于自定义和高效下单。
- **Classic TWS 经典界面**：保留了早期 TWS 风格的传统多窗口布局，适合习惯旧版本工作流的老用户。

!!! note "界面位置"
    TWS 主界面的产品定位示意可参考 IBKR 营销页的 [TWS 介绍图](https://www.interactivebrokers.com/en/trading/tws.php)，包含 Mosaic 与 Classic TWS 的典型布局示例。

## 关键要点

- **TWS 适合谁**：多产品、跨市场、有经验、有量化或主动交易需求的活跃交易者。
- **覆盖范围**：一个平台，160+ 全球市场，多种资产类别。
- **两种界面**：Mosaic（现代默认） / Classic TWS（传统布局），可在登录时或后续切换。
- **同类替代**：如果 TWS 桌面端对设备要求较高，盈透还提供 [Client Portal](https://ibkrguides.com/clientportal/home.htm)（网页端）、[IBKR Desktop](https://ibkrguides.com/ibkrdesktop/home.htm)（新一代桌面）、[IBKR Mobile](https://ibkrguides.com/iphone/home.htm)（手机 / 平板）等平台。
