# Client Portal API 文档翻译

> 英文原文：[https://www.interactivebrokers.com/api/doc.html](https://www.interactivebrokers.com/api/doc.html)

本目录翻译 Interactive Brokers Client Portal API 官方文档。

Client Portal API 是 IBKR 提供的 RESTful 接口，支持：

- 📊 账户与持仓查询
- 💹 行情数据订阅
- 📝 下单与撤单
- 📈 历史数据
- 🔐 通过 OAuth 1.0a / 2FA 鉴权
