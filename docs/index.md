# 盈透证券 API 中文文档

> Interactive Brokers（盈透证券）TWS API 与 Client Portal API 的中文翻译与实战指南

本项目由社区驱动，将 IBKR 官方英文技术文档翻译为简体中文，并对关键概念补充实战示例。

## 项目范围

| 模块 | 状态 | 说明 |
| --- | --- | --- |
| TWS API | ✅ 14 / 19 章节 | 盈透经典的桌面交易 API，覆盖下单、行情、账户等核心功能 |
| Client Portal API | ✅ 7 / 7 章节 | 盈透新一代基于 OAuth 的 REST API |
| 实战策略 | ✅ 3 个示例 | 基于 `ib_insync` 的量化示例：均线交叉 / 通道突破 / 网格交易 |

详细进度见各模块的 README 索引页。

## 阅读建议

- **初学者**：先看 [TWS API / Introduction](tws-api/introduction.md)，了解整体架构
- **实操派**：直接看示例代码（每个章节末尾的「实战示例」区块）
- **术语对照**：随时查阅 [术语表](glossary.md)
