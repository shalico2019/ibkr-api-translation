# Client Portal 使用手册

> 网页端操作指南（占位中，待社区贡献者启动翻译）

Client Portal 是盈透的官方网页交易终端，无需下载安装，浏览器打开即用。本模块将 [ibkrguides.com/clientportal](https://www.ibkrguides.com/clientportal/home.htm) 官方英文文档翻译为简体中文，聚焦**实操**而非概念解释。

!!! note "与 Client Portal API 模块的区别"
    本站另有 [**Client Portal API**](../client-portal-api/README.md) 模块，是面向**程序员**的 REST API 编程文档。本模块面向**交易员**，讲的是 Client Portal 网页本身的 UI 操作。两者互不重叠，请按需查阅。
